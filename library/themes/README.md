# Themes Library

Each `<name>.yaml` is one **themed world** — the setting / mood the
protagonist explores in this issue. Stores: theme_world description,
lighting_principles, default cover line template, and a per-page hint
list (length must match the layout's page_count).

When a free-form input run produces a new theme (user described one in
the request), the agent auto-persists a yaml here for re-use.

Note this layer does NOT contain full per-page prompts — those are
generated by the agent at Phase 3+4 from the simple-mode prompt template
(`references/simple-mode-prompts.md`) using the page_plan_hints as scene
guidance. Compared to a styled-mode "content" yaml (which would carry
full prompts), themes here are lighter (~25 lines vs ~440 lines).

See `library/SCHEMA.md` for the schema.
